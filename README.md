# Predictive Modeling and Risk Scoring for Bank Customer Churn

This project analyzes customer churn in the banking sector using an Excel-based analytical approach.

## Objective
To identify high-risk customers likely to churn using an explainable risk scoring framework.

## Methodology
- Data cleaning and preprocessing in Excel
- Exploratory analysis using pivot tables
- Risk feature engineering (Activity, Age, Product, Balance)
- Risk scoring and categorization (High / Medium / Low)
- Model validation using churn comparison

## Tools Used
- Microsoft Excel

## Note
This is an Excel-based analytics project focused on business interpretability rather than machine learning models.
